Class $W.Object

$W.Object(type, shouldAdd)
Parameters:
type
The type of rendering to use for this object, possible values are:
$W.GL.POINTS
$W.GL.LINES
$W.GL.LINE_LOOP
$W.GL.LINE_STRIP
$W.GL.TRIANGLES
$W.GL.TRIANGLE_STRIP
$W.GL.TRIANGLE_FAN
{Boolean} shouldAdd
Set to false to not add this the the object list in case you want to handle rendering in a specific manner, e.g. as the child of another object.
